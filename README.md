### 安装使用  
#### 通用安装
``安装方法: git clone https://github.com/dddinmx/Hound-Detector.git``  
``pip3 install -r requirements.txt``  
#### Mac
``可使用打包好的执行文件，使用前给执行权限``

#### 使用
``usage: fp.py [-h] [-u URL] [-f FILE]``  

  
- [x] 识别单个URL指纹
- [x] 识别多个URL，放进同一txt文件中，一行一个URL
- [ ] 输入IP自动探测Web应用并识别指纹，coming soon.
